# Jupyter

## Write script

`%%writefile foo.py`

## Display script

`%pycat foo.py`

## Run script

`%run foo.py`

## Benchmark script

`%%time`

## Show variable

`%who str`

## Show all variables

`%whos`

## Show variable info

`%pinfo foo`

## Store variable

`%store foo`

## Load variable

`%store -r foo`

## Render HTML

`%%html`

## Run JavaScript

`%%js`

## Set environment variable

`%env foo bar`

## Show all magic commands

`%lsmagic`
