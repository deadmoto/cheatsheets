# Numpy
`import numpy as np`

## Arrays

### Array creation

#### Array of constants
`np.array([0, 1, 2])`

#### Copy of array
`copy = np.copy(array)`

#### Array of zeros
`np.zeros(10)`

#### Array or matrix of ones
`np.ones(10)`  
`np.ones((3, 3))`

#### Range of values
`population = np.arange(1, 100)`

#### Linear space
`population = np.linspace(start=0, stop=41, num=42)`

#### Random sample
`sample = np.random.choice(population, 10)`
