# vim

## Editor

### Search and replace
`:%s/foo/bar/g`

### Sort
`:sort`

## Tabs

### New tab
`:tabnew <filename>`

### Next tab
`:tabNext`

## Split view

### Vertical split view
`vsp <filename>`

### Horizontal split view
`sp <filename>`

## Run

### Run current file
`:!%p`
or
`:! sh %`

### Run in new tab
`:tabdo <command>`

## Hotkeys
> v to select from cursor to EOL
> V to select line  
> D to delete  
> Y to copy
> p to paste before cursor  
> P to paste after cursor
