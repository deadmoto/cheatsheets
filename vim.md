# vim

## Editor

### Search and replace
`:%s/foo/bar/g`

### Sort
`:sort`

## Tabs

### New tab
`:tabnew <filename>`

### Next tab
`:tabnext` or `gt`

### Last tab
`:tabprevious` or `gT`

## Split view

### Vertical split view
`vsp <filename>`

### Horizontal split view
`sp <filename>`

## Run

### Run current file
`:!%p`
or
`:! sh %`

### Run in new tab
`:tabdo <command>`

## Shortcuts

### Navigation
> gg - move to beginning of file
> 0  - move to beginning of line
> $  - move to end of line 

### Selection
> v to select from cursor to EOL
> V to select line

### Editing  
>  D - delete
> dG - delete to EOF   
>  Y - copy
>  p - paste before cursor  
>  P - paste after cursor
